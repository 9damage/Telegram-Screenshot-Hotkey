# Telegram Screenshot Hotkey 3.0

## Основные изменения

- добавлена защищённая веб-галерея скриншотов;
- новые снимки появляются автоматически;
- добавлена маркировка непросмотренных снимков;
- реализовано одиночное, групповое и полное удаление;
- добавлены статистика клиента, очередь и занятое место;
- добавлен выбор автоочистки: 1, 3, 7, 14, 30, 90 или 365 дней;
- добавлена кнопка завершения процесса на устройстве;
- время скриншотов переведено на МСК;
- увеличены заголовки статистики и выровнен крестик просмотра;
- добавлено безопасное хранение метаданных в SQLite;
- клиент переведён на `https://api.shera2tap.ru`;
- старый адрес VDS автоматически заменяется на новый HTTPS API;
- добавлена идемпотентность загрузок через `X-Upload-ID`;
- обновлён экран входа и единый фирменный логотип;
- добавлен пароль администратора, CSRF-защита и ограничение попыток входа;
- обновлена сборка для Python 3.14 и PyInstaller 6.21.0;
- исправлено закрытие соединений SQLite;
- удалены устаревшие инструкции версии 2.0 и файл с реальными секретами.

## Совместимость

- существующий `config.json` можно оставить рядом с новым EXE;
- адрес `http://147.45.38.195/upload` будет заменён клиентом автоматически;
- серверные данные версии 2.0 сохраняются при обновлении.
