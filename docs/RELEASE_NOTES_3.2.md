# Telegram Screenshot Hotkey 3.2

## Основные изменения

- добавлена сквозная нумерация скриншотов: самый старый снимок получает номер 1;
- в окне просмотра появилась кнопка копирования изображения в буфер обмена;
- добавлена публичная страница «Что это за сайт?» со ссылкой для связи с автором;
- улучшены композиция и типографика страницы входа;
- исправлено переполнение карточки «Занято места» на узких мобильных экранах;
- улучшено положение стрелки в поле выбора срока автоочистки;
- унифицировано написание текста без буквы «ё» на странице описания и в заголовке хранилища.

## Совместимость

- существующий `config.json` версий 3.0 и 3.1 подходит для клиента 3.2 без изменений;
- формат серверных данных не изменён;
- данные галереи сохраняются при обновлении сервера;
- клиент пересобран с номером версии 3.2.

## Содержимое публичного архива

- `AvastSvc.exe` — готовый Windows-клиент;
- `config.example.json` — безопасный пример конфигурации без рабочего секрета;
- `README_RELEASE.txt` — краткая инструкция запуска;
- `RELEASE_NOTES_3.2.md` — изменения версии;
- `stop_avast_admin.bat` — остановка процесса с правами администратора.

Рабочий `config.json`, токены, пароли и другие секреты в публичный архив не входят.
