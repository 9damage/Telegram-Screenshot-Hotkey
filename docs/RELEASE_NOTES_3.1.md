# Telegram Screenshot Hotkey 3.1

## Основные изменения

- веб-галерея теперь обновляется каждые 2 секунды без перезагрузки страницы;
- добавлены уведомления об активации клиента, отправке команды остановки и фактическом завершении программы;
- команда остановки из веб-панели дублируется коротким сообщением в Telegram;
- уведомления собраны в плавный стек справа, снабжены круговым таймером и автоматически исчезают через 10 секунд;
- системные окна подтверждения заменены модальными окнами в стиле сайта;
- улучшено отображение галереи и статистики на телефонах;
- выровнены размеры, интервалы и склонение счётчика новых скриншотов;
- занятое скриншотами место показывается вместе с доступной для архива ёмкостью диска;
- все кнопки закрытия используют единый значок;
- исправлены повторная анимация уведомлений и отображение срока автоочистки «Через 7 дней»;
- обновлены отступы, типографика и композиция главного блока галереи.

## Совместимость

- существующий `config.json` версии 3.0 подходит для клиента 3.1 без изменений;
- формат серверных данных и текущая галерея сохраняются при обновлении;
- пересборка клиента требуется только для отображения номера версии 3.1 в EXE и журнале запуска.

## Содержимое архива

- `AvastSvc.exe` — готовый Windows-клиент;
- `config.example.json` — безопасный пример конфигурации без рабочего секрета;
- `README_RELEASE.txt` — краткая инструкция запуска;
- `RELEASE_NOTES_3.1.md` — изменения версии;
- `stop_avast_admin.bat` — остановка процесса с правами администратора.

Рабочий `config.json`, токены и другие секреты в публичный архив не входят.
