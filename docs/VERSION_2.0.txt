AvastSvc v2.0

Функции версии 2.0:
- отправка скриншотов по NumPad 0;
- локальная очередь и повторная отправка;
- работа через VDS relay;
- сообщение «Запуск успешен.»;
- команда /queue;
- команда /stop;
- сообщение «Программа успешно завершила работу.»;
- имя процесса AvastSvc.exe;
- иконка из client/icon.ico.

Примечание: в этой версии глобальная клавиша всё ещё реализована через pynput.
