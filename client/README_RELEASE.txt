Telegram Screenshot Hotkey 3.2

1. Переименуйте config.example.json в config.json.
2. Вставьте актуальный relay_secret в config.json.
3. Не меняйте адрес https://api.shera2tap.ru/upload.
4. Запустите AvastSvc.exe.
5. Для скриншота нажмите NumPad 0.

AvastSvc.exe и config.json должны находиться в одной папке.
Не публикуйте заполненный config.json: он содержит секрет доступа.
